# nao

nao presents itself as an open source analytics agent builder focused on context engineering for data teams.

## Positioning

- Build, evaluate, and deploy analytics agents on top of an existing data stack.
- Structure agent context like a file system rather than relying on generic retrieval alone.
- Offer a chat UI so non-technical users can query company data.

## Product claims highlighted on the homepage

- 100% open source.
- Bring your own model key, including providers like Claude, Gemini, GPT, Mistral, Bedrock, and Ollama.
- Ask questions from Slack or Teams.
- Self-hosting and security positioning, including a SOC 2 mention.
- Context engineering playbook and documentation as core educational material.

## Notable details

- The site links to the GitHub repository at `github.com/getnao/nao`.
- The structured data on the page describes both a free open source offer and a paid Pro seat priced at 30 USD.
- Messaging is aimed at analytics engineering, warehouse-aware SQL, and governed business context.
