## Overview

Moment describes itself as "a new type of collaborative workspace that makes it easy to build personalized software using AI" with built-in infrastructure for authentication, real-time editing, and API access.

## Key Vision

The platform aims to enable coding agents to create customized applications. Rather than building standalone enterprise applications, Moment focuses on team-oriented tools—dashboards, runbooks, and internal utilities that integrate live data and metrics.

## Four Core Features

**1. Markdown-Based Architecture**
Documents are stored as actual `.md` files in git repositories on users' disks, enabling integration with coding agents like Claude Code and Copilot CLI.

**2. Collaborative Editing**
The platform provides "high-performance, built-in, live collaborative editing" with claimed features including 60fps performance, offline editing capability, and incremental updates rather than full document replacements.

**3. Native Programmability**
Built-in React integration and Tailwind CSS support allow users to generate custom widgets and applications without extensive API layers.

**4. Secure API Access**
Moment includes a Postman-like interface for authenticated API requests, with credential proxying to avoid distributing API keys to clients. The platform supports Kubernetes integration with planned AWS support.

## Positioning

The article frames Moment as addressing limitations of pre-AI SaaS tools like Notion and Airtable, leveraging AI agents to generate application code on-demand.
