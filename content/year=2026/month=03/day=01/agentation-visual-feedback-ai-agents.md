# Agentation: Visual Feedback Tool for AI Agents

## Overview
Agentation is a desktop-only platform that transforms UI annotations into structured context for AI coding agents. It enables developers to click elements, add notes, and generate formatted output for use with Claude Code, Cursor, and similar AI tools.

## Key Features

**Core Functionality:**
- Click any UI element to create annotations
- Hover to see highlighted element names
- Export formatted markdown with CSS selectors and component names
- Integration with AI agents through copy-paste or MCP protocol

**What Agents Receive:**
The platform provides AI tools with "CSS selectors to grep your codebase, React component names to find the right file, computed styles to understand current appearance, and your feedback with intent and priority."

## Usage Workflow

1. Activate the toolbar (bottom-right corner)
2. Hover over elements to identify them
3. Click to add annotations
4. Write feedback and submit
5. Copy formatted output
6. Paste into your AI agent

## Integration Options

- **MCP Integration**: Skip copy-paste; agents access feedback directly
- **Webhooks**: Push annotations to external services
- **API**: Build custom integrations
